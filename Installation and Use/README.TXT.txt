These are the steps on how to run Stock Management System In PHP.

Step 1:Download and install local web serever
First download and Install any local web server such as XAMPP/WAMP.
Open your XAMPP/WAMP's Control Panel and start Apache and MySQL.

Step 2: Download source code.
Second, download the source code provided as the zipped file

Step 3: Extract file.
Third, after you have finished downloading the source code, extract the zip file.
stock management system zip file


Step 4: Copy project folder.
Forth, copy the project folder (Stock Management System) and paste it into the "htdocs" directory if you are using XAMP or  paste it into the "www" directoryif f you are using WAMP,

Step 5: Open xampp/wamp.
Fifth, open xampp/wamp and start the apache and MySQL.


Step 6: Open browser.
Sixth, Open a browser and go to URL “http://localhost/phpmyadmin/”


Step 7: Create database.
Seventh, click on databases tab and Create database naming “StockManager”.
stock management system create database

Step 7: Import “stock.sql”.
Seventh, Click on browse file and select “stock.sql” file which is inside “database” folder and after import click “go“.
stock management system import sql file

Step 8: Open browser and type folder name.
Eight, Open a browser and go to URL “http://localhost/Stock”.

Step 9: Explore manipulating.
